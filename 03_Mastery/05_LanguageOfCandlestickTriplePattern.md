# Morning star/ Evening star

![](../files/035-mastery.png)

![](../files/036-mastery.png)

```
For Morning Star,
Entry = (HH + LL)/ 2
R = HH - RR
T1 = E + R
T2 = T1 + R
SL = LL - 1

For Evening Star,

Entry = (HH + LL)/ 2
R = HH - RR
T1 = E - R
T2 = T1 - R
SL = HH + 1

```
